This is the official website for the Montana PM-2.5 Concentration Map project. Our goal is to create a seamless, PM-2.5 concentration map for Montana. Then, we aim to offer 3-day projections for those concentrations. This project is a joint partnership between Montana Tech and the University of Montan. We are funded by the Montana Idea Network for Biomedical Research Excellence (INBRE). Please visit out About page for more information.
